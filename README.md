# RAS Mains PYQ — Exam Pattern Study Dashboard

Interactive analytics over 1,003 RAS Mains previous-year questions (Papers 1–4),
compiled for exam preparation from publicly listed PYQs (source: rajras.in).

Features: subject/marks weightage, answer-length mix, year trends, subject×year
consistency heatmap, per-subject high-yield topic recurrence, a cross-subject
priority revision list (CSV export), and a searchable question bank.

Open `index.html` in any browser — fully self-contained (data embedded).
